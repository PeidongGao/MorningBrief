# Usage Disclaimer

Last reviewed: July 6, 2026

This repository provides a small local tool for running a configured
Codex-based Morning Brief workflow and browsing generated reports. The tool is
intended for low-volume, user-initiated personal review and record keeping.

MorningBrief does not include private prompts, notes, Codex session logs,
generated reports, local configuration, or source material. Users configure
their own local paths through `.env` or environment variables and are
responsible for deciding what files the configured workflow may read.

Generated reports may summarize, transform, or quote information from the
user's own local materials and from whatever sources their configured prompt
workflow accesses. Users are responsible for reviewing generated reports before
using, publishing, or redistributing them.
